Post processing of active sensing ro-bat in the arena with obstacles

Folders:

- analysis
    yaml documents containing the ground truth obstacle distance, the distance error, the ground truth angle and the angle error as lists generated by a blind tracking of the robot.
    
    File name structure is yyyymmdd_hh-mm-ss.yaml, which indicates the date and time in which the experiment was conducted.

- audio:
    eight-channel audio recorded by the robot during the run.
    File name structure is yyyymmdd_hh-mm-ss.wav
